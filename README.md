# Regional-Sales-Analysis
This project delivers an end-to-end sales analytics solution using 5 years of U.S. regional sales data. It integrates MySQL for data storage & querying, Python for Exploratory Data Analysis (EDA), and Power BI for interactive dashboards to uncover trends, profitability drivers, and regional growth opportunities.

#Business Objective:
  Identify seasonal sales patterns
  Analyze product, channel, and regional performance
  Highlight high-value customers and margin opportunities

#Tools & Technologies:
  MySQL – Data modeling, joins, aggregations
  Python (Pandas, Matplotlib) – Data cleaning & EDA
  Power BI – Interactive dashboards & visual insights
  Excel / CSV – Source data 

#Key Insights:
  Strong seasonality with peaks in May–June and lows in January
  Products 26 & 25 contribute ~25% of total revenue
  Wholesale drives 54% of sales volume; Export delivers highest margins (~38%)
  California is the top-performing state; the West region leads overall sales

#Deliverables:
  Cleaned and analysis-ready dataset
  MySQL analytical queries
  Interactive Power BI dashboard with filters by time, region, product, channel, and customer

#Conclusion:
  This project demonstrates a complete BI workflow from database design to business insights, enabling stakeholders to self-serve           analytics and make informed decisions.
